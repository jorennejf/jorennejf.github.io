DML SteamVR Home Environment

July 2017


This is one screenshot of how the room looks like in a virtual reality headset.

About

The DML SteamVR Home Environment is an interactive virtual reality environment based on UCSD's Digital Media Lab in Geisel Library. Large panels replace the television on the wall to allow users to view their friends, select a game, or move to a different room. Users can also teleport around the room and pick up the chairs or various objects on the shelves.

This was a summer project I did while working part-time at the DML. My manager wanted me to recreate the room to set up as our home environment whenever visitors use our virtual reality headsets.

    Tools

  • Google SketchUp
  • SteamVR Workshop

Users can point to these panels on the walls to access other features.

When the user extends their controller towards an interactive object, the object glows, indicating that it can be picked up.

This shelf is full of a few interactive objects that are actually on display in the real Digital Media Lab at UCSD as 3D printed objects.
