San Diego Chewsr

Feb 2022


Overview


Goal

The goal of this project was to solve the problem of figuring out what my partner and I want to eat in San Diego that we know we would like.


    My Roles

  • User Interface Design
  • Front-End Development

    Tools

  • Google Drawing
  • HTML
  • CSS
  • JavaScript

Story

For Valentine's Day, I wanted to make my partner a food randomizer web app that we can pull up on our phones whenever we're stuck on what to eat. Through the years, we already have a lot of restaurants we frequently visit, so I wanted to limit the options to food we know we like.


Design Process


    Design & Prototype

  1. Because the concept was so simple, I knew the UI design only needed one focal point: a button. However, I also wanted to include filters in case we had more specific cravings, so I incorporated that in a pop-up accessible through the main page's secondary button.
    Main Page
    Filters

    2. Develop

  2. For the development process, I wanted to focus on functionality first before design (I needed a working web app in time for Valentine's Day! I can always spruce up the design later since it's only for our personal use). This first version had the filters and randomizer all working properly, but I still had time to finalize the design.
    Main Page
    Results
    Filters

  3. For the design, I researched other simple apps to clean up the web app's main page, really focusing on that single button. Additionally, I decided to include the full list of restaurants I included in this web app, so that we as the users can see what our options are if we wanted to just pick from the list rather than use all the filters. All of these things were finalized during the development process of this project.
    Main Page
    Filters

Conclusion

Thankfully, I managed to finish working on everything I wanted to do a couple days before Valentine's Day. I almost got caught working on it too, but my partner was still pleasantly surprised! If we ever find a new restaurant we like, I can still add it to this web app. For now, there are plenty of options so that we're never faced with the problem of picking a restaurant ever again.
